# Product Goal

Liclick 3D Texture is a Web AI 3D Texture Studio for importing 3D assets, capturing current views, generating texture imagery, projecting generated images back onto models, managing layer stacks, baking UV textures, and exporting results to DCC tools.

The long-term product goal aligns with the broad workflow category represented by modern AI texture tools: Projects, Editor, Web3D viewport, reference images, AI generation, projected layers, inpaint, multiview, normal maps, segmentation, quick masks, layer stack, and export to GLB, Blender, and 3ds Max.

This implementation must remain clean-room:

- Do not copy private source code, bundles, CSS, icons, logo, images, product copy, private APIs, or brand assets from any competitor.
- Public screenshots and public feature descriptions may inform information architecture only.
- Visual identity, component implementation, data contracts, and assets are Liclick-owned or open-source.

## Four-Week MVP Scope

Week 1 builds the UI shell, Web3D viewport, primitive/default model, GLB import interface, reference images, and panel structure.

Week 2 adds capture passes for color, mask, depth, and normal, plus the Liclick API mock adapter.

Week 3 adds projected layer preview, layer panel workflows, and camera recall metadata.

Week 4 adds UV bake MVP, basecolor export, and demo polish.
